# Bell-Voucher-Calculator

<img width="559" height="500" alt="The application interface, with its title, a text box to enter the amount of miles you have, a calculate button and a reset button. An exit button sits at the bottom." src="https://github.com/user-attachments/assets/634bb965-4640-497f-a5e5-104395e2287e" />

An extremely simple calculator for determining how many bell vouchers a certain amount of miles will get you, and the total amount of bells you would get from those vouchers. I made it for Animal Crossing: New Horizons.

Suggestions and improvements are welcome! This is my first github project, so I want to get better are making software.
